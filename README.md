# Debian Tuning

Debian 12 一键优化脚本，专为 sing-box 代理节点设计。

## 快速开始
# 方式1：分步执行（推荐）
curl -fsSL https://raw.githubusercontent.com/BeiFang798/vps-optimizer/main/tuning-debian12-cn.sh | bash
reboot

# 方式2：先下载再执行
curl -fsSL https://raw.githubusercontent.com/BeiFang798/vps-optimizer/main/tuning-debian12-cn.sh -o /root/tuning.sh
bash /root/tuning.sh
reboot
